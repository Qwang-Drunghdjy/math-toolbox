# 求和方法

## 级数

### $p$-级数和几何级数

> **定义: ($p$-级数)** 形如
>
> ```math
> \sum_{n = 1}^{\infty} \frac{1}{n^p}
> ```
>
> 的级数.

**$p$-级数的敛散性:**

- 当 $p \leq 1$, $p$-级数发散.
- 当 $p > 1$, $p$-级数收敛.

> **定义: (几何级数)** 形如
>
> ```math
> \sum_{n = 1}^{\infty} aq^{n}, \quad a \neq 0
> ```
>
> 的级数.

**几何级数的敛散性:**

- 当 $|q| < 1$, 几何级数收敛.
- 当 $|q| \geq 1$, 几何级数发散.

## 等差数列和等比数列求和

**等差数列前 $n$ 项求和公式:**

```math
S_n = \frac{1}{2}n(a_1 + a_n) = na_1 + \frac{n}{2}(n - 1)d
```

- 求和依赖项数 $n$ 的明确, 不适用于无穷项求和.

**等比数列前 $n$ 项求和公式:**

```math
S_n =
\begin{cases}
    \frac{a_1(1 - q^n)}{1 - q}, & (q \neq 1) \\
    na_1, & (q = 1)
\end{cases}
```

无穷等比数列求和公式:

```math
S_\infty = \frac{a_1}{1 - q}
```

- 其中 $q$ 是公比, 且 $|q| < 1$. 如果 $|q| \geq 1$, 等比数列的和是发散的.
