# 组合数学

## 排列{#permutation}

```math
    P_n^k = \frac{n!}{(n-k)!}
```

从 $n$ 个元素中取出 $k$ 个元素, $k$ 个元素的排列数量.

排列需要考虑元素的顺序.

- 排列计算假设元素不同且不重复.

## 组合{#combination}

```math
    C_n^k = \frac{P_n^k}{k!} = \frac{n!}{k!(n-k)!}
```

从 $n$ 个元素中取出 $k$ 个元素, $k$ 个元素的组合数量.

组合不考虑元素的顺序.

- 组合计算假设元素之间独立, 且选择无偏好.
